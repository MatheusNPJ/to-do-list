# 📝 Task List (Terminal App)

Simple task list system built using terminal commands.  
Developed by **Matheus Navalski**, **Pedro Valeriano**, and **Eduardo Leal**.

## 🚀 Features

- Add tasks  
- List tasks  
- Mark tasks as done  
- Remove tasks  
- Data saved in a text file

## 📦 How to Use

```bash
chmod +x tasks.sh       # Give execution permission
./tasks.sh add "Task"   # Add task
./tasks.sh list         # List tasks
./tasks.sh done 1       # Mark as done
./tasks.sh remove 1     # Remove task
